# Llanquihue Tour App

Proyecto Java desarrollado para la actividad de la Semana 4 de Desarrollo Orientado a Objetos I.

## Descripción

La aplicación permite leer información de tours turísticos desde un archivo `tours.txt`, crear objetos de tipo `Tour`, almacenarlos en un `ArrayList` y filtrar los tours según su precio.

## Estructura de carpetas

LlanquihueTourApp/
├── src/
│   ├── model/
│   │   └── Tour.java
│   ├── data/
│   │   └── GestorDatos.java
│   └── ui/
│       └── Main.java
├── resources/
│   └── tours.txt
└── README.md

## Cómo ejecutar

1. Abrir el proyecto en IntelliJ IDEA.
2. Verificar que el archivo `tours.txt` esté dentro de la carpeta `resources`.
3. Ejecutar la clase `Main`, ubicada en el paquete `ui`.
